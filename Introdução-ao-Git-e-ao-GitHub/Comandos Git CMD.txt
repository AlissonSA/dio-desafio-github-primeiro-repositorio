Comandos Básicos Para Navegar em um Terminal:

dir -  Listar
cd / (cd Windows) - Entrar pasta
cd .. - Retroceder pastas
cls - Limpar a tela (o terminal Windows)
mkdir - Criar pasta
echo > - Cria um output (um arquivo)
del - deletar arquivos (conteúdo da pasta, apenas e não uma pasta/repositório)
rmdir - Remover Diretório
/S /Q - Flex (???)
-a - flag


Git Bash é um terminal estendido otimizado para o Git
